# CodeQuest 🚀

> 단계별로 배우는 멀티언어 코딩 학습 플랫폼

## ✨ 주요 기능

- 🌱 **체계적 학습**: Hello World부터 Big Tech 인터뷰 문제까지 13단계
- 🌍 **8개 언어**: Python, JavaScript, TypeScript, Java, C++, Go, Rust, ABAP
- 💡 **완벽한 힌트**: 모든 언어에 대한 맞춤형 힌트 제공
- ⭐ **Default 언어**: 자주 쓰는 언어를 default로 설정 (☆ 버튼)
- 🎨 **멀티 에디터**: 한 문제를 여러 언어로 동시에 작성
- 📊 **진행 추적**: 자동 저장 및 통계 (Streak, Points)
- 🎨 **차분한 디자인**: 하늘색, 살구색, 베이지색 조합

## 🚀 빠른 시작

1. **index.html** 다운로드
2. GitHub 저장소에 업로드
3. Settings > Pages에서 활성화
4. `https://your-username.github.io/codequest` 접속!

## 🎯 사용 방법

### Default 언어 설정
- 언어 탭 옆의 ☆ 클릭 → default 설정
- ★으로 표시되며 첫 탭으로 이동
- 설정 자동 저장

### 문제 풀기
1. 왼쪽에서 카테고리 선택
2. 문제 읽고 💡 힌트 확인
3. 코드 작성 (자동 저장)
4. Run Code → Submit

## 🎓 학습 단계

1-5단계 (기초): Variables, Operators, Conditions, Loops, Functions
6-8단계 (중급): Arrays, Strings, Data Structures
9-11단계 (고급): Algorithms, OOP
12-13단계 (실전): Real-World, Big Tech Interviews
